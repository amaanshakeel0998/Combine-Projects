# Projects Dashboard Portfolio

A single-page portfolio dashboard with three detailed project pages, showcasing academic scheduling and building management systems. The UI includes a particle background canvas, custom cursor, animated project cards, and a team section toggle.

## Developer

Muhammad Amaan

## What's Inside

- `index.html`: Main dashboard with project cards and a team toggle.
- `project-1.html`: Exam Scheduling System project details.
- `project-2.html`: Lecture Scheduling System project details.
- `project-3.html`: Building Handling System project details.
- `style.css`: Shared styling for all pages.
- `script.js`: Canvas background, custom cursor, dropdown, and UI interactions.

## Features

- Animated multi-layer particle background with mouse interactions.
- Custom cursor with hover scaling and click feedback.
- Responsive layout for desktop and mobile.
- Project detail pages with consistent visual styling.
- Dropdown navigation for quick project access.

## Running Locally

Open `index.html` in a browser. No build step required.
