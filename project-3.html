Overview

The Building Handling System is an innovative web-based application that provides a visual 3D representation of campus buildings, floors, and rooms. This system combines spatial management with scheduling functionality, allowing administrators to manage room allocations, track capacity, and visualize building layouts in an intuitive interface.

Visual 3D Representation

Interactive Building Models

Buildings are rendered in 3D with the ability to rotate, zoom, and explore each structure. The visual representation helps administrators understand spatial relationships and building layouts.

Floor Visualization

Each floor can be viewed independently with its room layout. This makes it easy to see available spaces and plan room assignments effectively.

Progress Tracking

Real-time progress indicators show the construction status of buildings as data is being processed, providing visual feedback during the generation process.
