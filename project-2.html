Academic Timetable Generator

January 10, 2026 System Architecture

The Academic Timetable Generator is a robust Flask-based application designed to automate the creation of complex lecture schedules while maintaining strict academic constraints.

Exhaustive Scheduling

The system utilizes a deterministic, exhaustive scheduling algorithm that validates schedules against large datasets. It ensures that teachers, subjects, and classrooms are allocated optimally without any overlapping conflicts.

Multi-Format Exports

One of its core strengths is the ability to generate professional-grade documentation. The system can export schedules directly to Excel (.xlsx) using OpenPyXL and PDF using ReportLab, complete with custom color-coding for different subjects.

Technical Stack

  • • Python (Flask)
  • • ReportLab (PDF Generation)
  • • OpenPyXL (Excel Processing)
  • • In-memory Session Storage
